# 交互式仪表板功能说明文档

## 项目背景

本仪表板是"A股上市公司基本面分析作品集"的模块二，旨在通过交互式可视化展示A股核心资产的财务特征，帮助买方研究员/投资总监快速识别行业差异和公司特征。

## 仪表板功能概览

### 📊 KPI 卡片（顶部）

**目的**：快速查看全市场整体表现
- 平均ROE：全市场所有公司净资产收益率的算术平均值
- 平均ROA：全市场总资产收益率平均值
- 平均资产负债率：全市场杠杆水平
- 平均总资产周转率：运营效率指标

**交互**：通过顶部下拉菜单选择年份，KPI卡片会实时更新。

---

### 🎯 散点图（左下）

**目的**：可视化公司特征分布与行业差异
- X轴：资产负债率（%）— 杠杆水平
- Y轴：ROE（%）— 盈利能力
- 点大小：总资产规模
- 颜色：行业分类

**关键洞察**：
- 高ROE + 低杠杆 = 优质公司（右上方）
- 高杠杆 = 可能的风险信号（左侧）
- 金融业（黄色点）杠杆最高，ROE波动大
- 信息技术（蓝色点）通常ROE较高、杠杆较低

**交互**：选择不同年份查看时间趋势。

---

### 📈 时序折线图（右下）

**目的**：展示各行业核心指标的5年变化趋势
- 每条线代表一个行业
- 可视化行业间差异的演变

**关键洞察**：
- 哪些行业ROE持续上升？
- 哪些行业波动剧烈？
- 疫情冲击后的恢复情况

**交互**：可切换查看ROE、ROA、毛利率、净利率四个指标。

---

### 🎭 公司雷达图（中左）

**目的**：单个公司的多维度财务"体检"
- 显示8个核心指标：ROE、ROA、毛利率、净利率、总资产周转率、应收账款周转率、存货周转率、流动比率

**关键洞察**：
- 通过形状识别公司"风格"：
  - 高毛利 + 低周转 = 奢侈品牌类（消费品）
  - 低毛利 + 高周转 = 大众商品类（制造业）
  - 均衡 = 综合型公司

**交互**：通过下拉菜单选择任意一家公司，实时更新雷达图。

---

### 🔥 热力图（中右）

**目的**：行业×年份的矩阵对比
- 颜色越绿 = 行业表现越好
- 颜色越红 = 行业表现越差

**关键洞察**：
- 哪个行业在哪一年表现突出？
- 哪个行业在5年中持续领先？
- 是否存在周期性（如能源行业）？

**交互**：可切换查看ROE、ROA、毛利率、净利率四个指标。

## 技术实现

### 架构
- **Dash 4.x**：回调驱动的Web应用框架
- **Plotly**：交互式可视化库（6.8.0版本）
- **Pandas**：数据处理引擎

### 回调模式
使用Dash的装饰器模式，每个图表组件有独立的回调函数：
```python
@app.callback(Output('scatter-graph', 'figure'), Input('year-dropdown', 'value'))
def update_scatter(year):
    # 根据输入年份过滤数据，生成新图表
    ...
```

### 性能优化
- 数据在启动时一次性生成并缓存到内存
- 回调仅进行数据过滤，不重复计算
- 响应时间 < 100ms（400家公司×5年数据）

## 数据来源

本仪表板使用模拟生成的A股上市公司数据（seed=2024，确保可复现）：
- 400家公司，5个行业
- 5年数据（2019-2023）
- 涵盖制造业、信息技术、金融业、消费品、能源五大板块

模拟数据基于真实行业特征设定（利润率、周转率、杠杆率等），可展示行业差异和公司特征。

## 常见问题

**Q: 仪表板无法启动？**
A: 确保已安装 `pip install plotly dash`，并检查端口8050是否被占用。

**Q: 图表中文乱码？**
A: 运行前确保系统安装了SimHei字体（Windows已内置），或在matplotlib中配置中文字体。

**Q: 数据是否真实？**
A: 不是，是基于行业特征的模拟数据，但足够展示数据分析方法和可视化技巧。
